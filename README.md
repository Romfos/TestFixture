# Description
Small experemental fixture library for unit tests

# Example
```csharp

var fixture = new Fixture();
var array = fixture.Create<string[]>();

```

# Supported runtimes

- .NET 6+
- .NET Framework 4.6.2+
- .NET Standard 2.0 (for some older runtimes)
